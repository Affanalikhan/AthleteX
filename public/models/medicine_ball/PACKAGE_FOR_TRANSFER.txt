INSTALLATION ON NEW SYSTEM
================================================================================

1. Extract ZIP file
2. Open terminal in extracted folder
3. Run: pip install -r requirements.txt
4. Run: python train_production.py (to train)
5. Run: python analyze_video_production.py video.mp4 (to analyze)

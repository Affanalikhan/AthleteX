STEP 1: Install & Start Backend
--------------------------------
Open terminal/command prompt:

cd backend
pip install -r requirements.txt
python main.py

‚úì Backend will start on http://localhost:8000

STEP 2: Install & Start Frontend
---------------------------------
Open a NEW terminal/command prompt:

cd frontend
npm install
npm start

‚úì Frontend will open automatically at http://localhost:3000

DONE! Upload a running video and click "Analyze Run"

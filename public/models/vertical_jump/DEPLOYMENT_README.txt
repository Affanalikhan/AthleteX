
# Vertical Jump Coach - Deployment Package

## Quick Start

1. Extract this zip file
2. Install dependencies: `pip install -r requirements.txt`
3. Train model (if not included): `python train_improved.py`
4. Start application: `python start_fixed.py`
5. Open browser: http://localhost:8080
